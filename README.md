# 中国古代建筑结构对比可视化项目

## 项目概述
本项目通过数据可视化技术，对比展示中国古代建筑两种主要结构形式——抬梁式与穿斗式的性能特点、材料使用和适用场景。

## 技术栈
- **前端框架**: 原生 HTML + JavaScript
- **可视化库**: ECharts 5.x (百度开源，CDN引入)
- **样式**: 自定义 CSS
- **数据存储**: JSON 格式

## 快速开始

### 1. 直接运行
双击 `index.html` 文件，在浏览器中打开即可。

### 2. 使用本地服务器（推荐）
```bash
# Python 3
python -m http.server 8000

# Node.js (需要安装 http-server)
npx http-server
```
然后访问 http://localhost:8000

## 核心图表说明

### 1. 雷达图 - 结构性能对比
- **指标**: 空间利用率、抗震性、施工难度、造价成本、用材量
- **对比**: 抬梁式 vs 穿斗式

### 2. 柱状图 - 木材性能对比
- **指标**: 抗压强度、抗弯强度、弹性模量
- **木材种类**: 楠木、松木、杉木、榆木

### 3. 饼图 - 构件尺寸对比
- **抬梁式**: 柱高、梁长、柱径分布
- **穿斗式**: 柱高、梁长、柱径分布

### 4. 散点图 - 适用场景分析
- **横轴**: 建筑高度
- **纵轴**: 跨度大小
- **点大小**: 用材量

## 数据来源说明

本项目数据参考以下文献：
1. 《营造法式》（李诫，宋代）
2. 故宫博物院建筑测绘资料
3. 四川传统民居调研数据
4. 《中国建筑史》（梁思成）
5. 相关学术论文（见 docs/数据来源.md）

**注意**: 本项目数据为教学演示用途，实际工程数据可能因具体建筑而异。

## 常见问题

### Q: 图表不显示？
A: 确保网络连接正常，ECharts 库通过 CDN 加载。可下载到本地使用。

### Q: 如何修改数据？
A: 编辑 `js/data.js` 文件中的数据数组。

### Q: 如何添加新图表？
A: 在 `js/charts.js` 中添加新的图表配置函数。
